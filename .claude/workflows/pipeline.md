<workflow_manifest>
Autonomous Bootstrapping Pipeline
</workflow_manifest>

<stages>
[PHASE 1] -> `spec.json` generated by Requirements Agent
   ↓ 
[PHASE 2] -> `architecture.json` generated by Architect Agent
   ↓
[PHASE 3] -> PARALLEL EXECUTION: 
             ├─> Database Agent (schema.prisma, migrations)
             ├─> Backend Agent (express app, routes, auth)
             └─> Frontend Agent (vite app, components)
   ↓
[PHASE 4] -> Review Agent parses full codebase against skills. 
             If failure -> Command Build Agents to repair.
             If success -> Proceed.
   ↓
[PHASE 5] -> Assembler Agent links frontend/backend, adds docker-compose, pushes to github.
</stages>

<enforcement>
No phase may be skipped. Artifact handoffs must occur exactly as defined.
</enforcement>
