<skill_context>
Frontend UI/UX Standards
</skill_context>
<rules>
- Components must be functional and use Hooks exclusively.
- Use Tailwind CSS for 100% of styling. Avoid custom CSS files.
- Ensure all asynchronous buttons display a loading spinner while fetching.
- Handle empty states gracefully (e.g. "No tasks found, create one!").
- Enforce strict accessible HTML (ARIA labels, semantic tags like `<main>`, `<nav>`, `<button>`).
</rules>
